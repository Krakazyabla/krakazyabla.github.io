Education

Projects

Examples of design, implemented with basic web technologies. Most of my projects are responsive, creating by method “mobile first". It’s a base for any site, that can be expanded and improved with CSS-frameworks: famous and widespread Twitter Bootstrap, minimalistic Pure.css, more advanced Foundation, simple and perspective Bulma, and other.

JavaScript can provide not only interactivity for elements of web-pages, but also whole set of tools for creating big web apps and, of course, browser games. Visual representation of games has been created with HTML + CSS.

React.js takes special place among developer’s tools. It’s a JavaScript library for building user interfaces. It is maintained by Facebook and a huge community of individual developers and companies. Functionality of my apps includes asynchronous fetching of data, working with local storage and service workers.

It's a single-page application that demonstrates building application with React, fetching data with 3rd party API - Google maps API and Media Wiki API. This app allows to explore with the map main mountains of the world - Himalayas. Clicking on any marker or list item shows wikipedia info about chosen mountain.
